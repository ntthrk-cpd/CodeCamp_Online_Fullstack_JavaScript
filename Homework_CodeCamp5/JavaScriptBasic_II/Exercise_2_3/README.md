# Exercise 2 #3 : วงวน for และ while

**แบบฝึกหัด**

    3. code ทั้งสองอันนี้จะแสดง alert ออกมาเหมือนกันทั้งหมดหรือไม่

```javascript
    for (let i = 0; i < 5; i++) alert( i );
```
```javascript
    for (let i = 0; i < 5; ++i) alert( i );
```
