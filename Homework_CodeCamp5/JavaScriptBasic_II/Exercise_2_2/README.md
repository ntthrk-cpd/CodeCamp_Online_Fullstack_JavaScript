# Exercise 2 #2 : วงวน for และ while

**แบบฝึกหัด**
    
    2. code ทั้งสองอันนี้จะแสดง alert ออกมาเหมือนกันทั้งหมดหรือไม่

```javascript
    let i = 0;
    while (++i < 5) alert( i );
```

```javascript
    let i = 0;
    while (i++ < 5) alert( i );
```
