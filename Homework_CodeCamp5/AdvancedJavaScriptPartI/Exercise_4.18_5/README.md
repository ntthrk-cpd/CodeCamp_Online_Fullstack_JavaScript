4.18.  แบบฝึกหัด

5.	จงเขียนฟังก์ชัน multiplyNumeric(obj, times) โดยถ้า properties นั้นมี value เป็น number ให้คุณ value นั้นด้วย times ถ้าข้อมูลเเป็นอย่างอื่นไม่ต้องทำอะไร

```javascript
// before the call
let menu = {
  width: 200,
  height: 300,
  title: "My menu"
};
```
```javascript
multiplyNumeric(menu, 2);

// after the call
menu = {
  width: 400,
  height: 600,
  title: "My menu"
};
```
