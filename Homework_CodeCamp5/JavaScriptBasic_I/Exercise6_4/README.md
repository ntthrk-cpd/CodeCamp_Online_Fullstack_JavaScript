# Exercise 6 #4 : การเขียนเงื่อนไข

### แบบฝึกหัด
>
    4.	เปลี่ยน if-else ข้างล่างในอยู่ในรูปของ Ternary Operators

```javascript
let age = prompt('How old are you?');
let price;
if (age < 18) {
    price = 2000;
} else {
    price = 3500;
}
```
