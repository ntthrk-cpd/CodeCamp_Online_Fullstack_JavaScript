2. DIFFERENTIATION



Tag ความเหมือน ความแตกต่าง
<hr>& <br> ใช้จัดระเบียบหน้าเว็บ
  • <br> ใช้สำหรับขึ้นบรรทัดใหม่
  • <hr> ใช้แสดงเส้นขีดคั่นทางแนวนอน
<ul>& <ol> ใช้สำหรับแสดงรายการ
  • <ul>ใช้สำหรับแสดงรายการแบบไม่มีลำดับ
  • <ol>ใช้สำหรับแสดงรายการที่มีลำดับ
<select>& <checkbox> &<radio button>
  • ใช้สำหรับสร้างรายการตัวเลือก
  • Select และ checkbox สามารถเลือกได้เพียงค่าเดียว
  • checkbox และ rabio button แสดงรายการให้เลือกเหมือนกัน
  • checkbox แสดงข้อมูลเป็นรายการ โดยสามารถติ๊กเลือกได้หลายรายการ
  • radio button และ checkbox แสดงข้อมูลเป็นลิสต์รายการ ต่างจาก select แสดงรายการให้เลือกแบบ drop-down
<div>& <span> ใช้กำหนด layout ของเอกสาร div จะมีลักษณะเป็น block element ทำให้ขึ้นบรรทัดใหม่เสนอ แต่ span จะเป็นลักษณะเป็น inline element จะไม่ขึ้นบรรทัดใหม่ จึงมักจะใช้ครอบส่วนที่เป็น text เพื่อกำหนด style
<script>& <style> ใช้กำหนดรูปแบบและลักษณะในหน้าเว็บ
  • script ใช้เพื่อกำหนดสคริปต์ฝั่งไคลเอ็นต์ (JavaScript)
  • style ใช้เพื่อกำหนดข้อมูลสไตล์สำหรับเอกสาร HTML
